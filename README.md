最近电影天堂的广告越来越多！！！还都少儿不宜！！！上学的时候写的一个爬虫，额命名可能不规范轻喷嘻嘻到时候再重构一下，搜索电影模块挂掉了我打算爬那个飘花电影，那个没验证电影还挺多！嗯等后续闲下来会写。现在凑活没事拿迅雷看个电影还不错~~~
